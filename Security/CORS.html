CORS

Cross-Origin Resource Sharing (CORS) là một cơ chế cho phép các tài nguyên được yêu cầu từ một domain khác. Bên trong, Nest sử dụng package Express cors hoặc Fastify @fastify/cors tùy thuộc vào platform bên dưới. Các package này cung cấp nhiều option mà bạn có thể tùy chỉnh dựa trên yêu cầu của mình.

Getting started #

Để bật CORS, gọi method enableCors() trên Nest application object.

const app = await NestFactory.create(AppModule);
app.enableCors();
await app.listen(process.env.PORT ?? 3000);

Method enableCors() nhận một configuration object argument tùy chọn. Các property có sẵn của object này được mô tả trong tài liệu chính thức của CORS. Một cách khác là truyền một callback function cho phép bạn định nghĩa configuration object một cách bất đồng bộ dựa trên request (on the fly).

Ngoài ra, có thể bật CORS thông qua options object của method create(). Đặt property cors thành true để bật CORS với cài đặt mặc định. Hoặc, truyền một CORS configuration object hoặc callback function làm giá trị property cors để tùy chỉnh hành vi của nó.

const app = await NestFactory.create(AppModule, { cors: true });
await app.listen(process.env.PORT ?? 3000);
