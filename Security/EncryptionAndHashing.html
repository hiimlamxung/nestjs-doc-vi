Encryption and Hashing

Encryption là quá trình mã hóa thông tin. Quá trình này chuyển đổi biểu diễn gốc của thông tin, được gọi là plaintext, thành một dạng thay thế được gọi là ciphertext. Lý tưởng nhất, chỉ các bên được ủy quyền mới có thể giải mã ciphertext trở lại plaintext và truy cập thông tin gốc. Encryption tự nó không ngăn chặn sự can thiệp nhưng từ chối nội dung có thể hiểu được đối với kẻ chặn bắt. Encryption là một hàm hai chiều; những gì được encrypt có thể được decrypt với key phù hợp.

Hashing là quá trình chuyển đổi một key đã cho thành một giá trị khác. Một hash function được sử dụng để tạo ra giá trị mới theo một thuật toán toán học. Một khi hashing đã được thực hiện, việc đi ngược từ output về input là không thể.

Encryption #

Node.js cung cấp một crypto module tích hợp sẵn mà bạn có thể sử dụng để encrypt và decrypt string, number, buffer, stream, và nhiều hơn nữa. Bản thân Nest không cung cấp thêm bất kỳ package nào trên module này để tránh đưa vào các abstraction không cần thiết.

Ví dụ, hãy sử dụng thuật toán AES (Advanced Encryption System) 'aes-256-ctr' với chế độ encryption CTR.

import { createCipheriv, randomBytes, scrypt } from 'node:crypto';
import { promisify } from 'node:util';

const iv = randomBytes(16);
const password = 'Password used to generate key';

// The key length is dependent on the algorithm.
// In this case for aes256, it is 32 bytes.
const key = (await promisify(scrypt)(password, 'salt', 32)) as Buffer;
const cipher = createCipheriv('aes-256-ctr', key, iv);

const textToEncrypt = 'Nest';
const encryptedText = Buffer.concat([
  cipher.update(textToEncrypt),
  cipher.final(),
]);

Bây giờ để decrypt giá trị encryptedText:

import { createDecipheriv } from 'node:crypto';

const decipher = createDecipheriv('aes-256-ctr', key, iv);
const decryptedText = Buffer.concat([
  decipher.update(encryptedText),
  decipher.final(),
]);

Hashing #

Đối với hashing, chúng tôi khuyến nghị sử dụng package bcrypt hoặc argon2. Bản thân Nest không cung cấp thêm bất kỳ wrapper nào trên các module này để tránh đưa vào các abstraction không cần thiết (giúp rút ngắn learning curve).

Ví dụ, hãy sử dụng bcrypt để hash một password ngẫu nhiên.

Đầu tiên cài đặt các package cần thiết:

$ npm i bcrypt
$ npm i -D @types/bcrypt

Sau khi cài đặt hoàn tất, bạn có thể sử dụng hàm hash, như sau:

import * as bcrypt from 'bcrypt';

const saltOrRounds = 10;
const password = 'random_password';
const hash = await bcrypt.hash(password, saltOrRounds);

Để tạo một salt, sử dụng hàm genSalt:

const salt = await bcrypt.genSalt();

Để so sánh/kiểm tra một password, sử dụng hàm compare:

const isMatch = await bcrypt.compare(password, hash);

Bạn có thể đọc thêm về các hàm có sẵn tại đây.
