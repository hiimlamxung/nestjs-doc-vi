Pipes

Không có sự khác biệt cơ bản nào giữa pipes thông thường và microservices pipes. Điểm khác biệt duy nhất là thay vì throw HttpException, bạn nên sử dụng RpcException.

Gợi ý

Class RpcException được import từ package @nestjs/microservices.

Binding pipes #

Ví dụ sau sử dụng pipe được khởi tạo thủ công với phạm vi method-scoped. Giống như với các ứng dụng dựa trên HTTP, bạn cũng có thể sử dụng controller-scoped pipes (tức là đặt prefix cho controller class với decorator @UsePipes()).

@UsePipes(new ValidationPipe({ exceptionFactory: (errors) => new RpcException(errors) }))
@MessagePattern({ cmd: 'sum' })
accumulate(data: number[]): number {
  return (data || []).reduce((a, b) => a + b);
}

accumulate(data) {
