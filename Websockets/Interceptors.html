Interceptors

Không có sự khác biệt nào giữa interceptors thông thường và web sockets interceptors. Ví dụ sau đây sử dụng một interceptor method-scoped được khởi tạo thủ công. Giống như với các ứng dụng dựa trên HTTP, bạn cũng có thể sử dụng gateway-scoped interceptors (tức là, prefix class gateway với decorator @UseInterceptors()).

@UseInterceptors(new TransformInterceptor())
@SubscribeMessage('events')
handleEvent(client: Client, data: unknown): WsResponse<unknown> {
  const event = 'events';
  return { event, data };
}
@UseInterceptors(new TransformInterceptor())
@SubscribeMessage('events')
handleEvent(client, data) {
  const event = 'events';
  return { event, data };
}
