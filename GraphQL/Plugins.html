Plugins

Plugins với Apollo #

Plugins cho phép bạn mở rộng chức năng cốt lõi của Apollo Server bằng cách thực hiện các thao tác tùy chỉnh để phản hồi các sự kiện nhất định. Hiện tại, các sự kiện này tương ứng với các giai đoạn riêng lẻ trong vòng đời request GraphQL, và với việc khởi động của chính Apollo Server (đọc thêm tại đây). Ví dụ, một plugin logging cơ bản có thể ghi lại chuỗi query GraphQL liên kết với mỗi request được gửi đến Apollo Server.

Custom plugins

Để tạo một plugin, hãy khai báo một class được gắn decorator @Plugin được export từ package @nestjs/apollo. Ngoài ra, để có autocompletion code tốt hơn, hãy implement interface ApolloServerPlugin từ package @apollo/server.

TypeScript
import { ApolloServerPlugin, GraphQLRequestListener } from '@apollo/server';
import { Plugin } from '@nestjs/apollo';

@Plugin()
export class LoggingPlugin implements ApolloServerPlugin {
  async requestDidStart(): Promise<GraphQLRequestListener<any>> {
    console.log('Request started');
    return {
      async willSendResponse() {
        console.log('Will send response');
      },
    };
  }
}

Với điều này, chúng ta có thể đăng ký LoggingPlugin như một provider.

TypeScript
@Module({
  providers: [LoggingPlugin],
})
export class CommonModule {}

Nest sẽ tự động khởi tạo plugin và áp dụng nó vào Apollo Server.

Sử dụng external plugins

Có một số plugin được cung cấp sẵn. Để sử dụng một plugin hiện có, chỉ cần import nó và thêm vào mảng plugins:

TypeScript
GraphQLModule.forRoot({
  // ...
  plugins: [ApolloServerOperationRegistry({ /* options */})],
}),
Gợi ý

Plugin ApolloServerOperationRegistry được export từ package @apollo/server-plugin-operation-registry.

Plugins với Mercurius #

Một số Fastify plugin hiện có dành riêng cho mercurius phải được load sau mercurius plugin (đọc thêm tại đây) trong cây plugin.

Cảnh báo

mercurius-upload là một ngoại lệ và nên được đăng ký trong file chính.

Để thực hiện điều này, MercuriusDriver cung cấp một tùy chọn cấu hình plugins tùy chọn. Nó đại diện cho một mảng các object gồm hai thuộc tính: plugin và options của nó. Do đó, việc đăng ký cache plugin sẽ trông như sau:

TypeScript
GraphQLModule.forRoot({
  driver: MercuriusDriver,
  // ...
  plugins: [
    {
      plugin: cache,
      options: {
        ttl: 10,
        policy: {
          Query: {
            add: true
          }
        }
      },
    }
  ]
}),
