Server-Sent Events

Server-Sent Events (SSE) là một công nghệ server push cho phép client nhận các cập nhật tự động từ server thông qua kết nối HTTP. Mỗi notification được gửi dưới dạng một khối văn bản kết thúc bằng một cặp dòng mới (tìm hiểu thêm tại đây).

Usage #

Để bật Server-Sent Events trên một route (route được đăng ký trong một controller class), annotate method handler với decorator @Sse().

@Sse('sse')
sse(): Observable<MessageEvent> {
  return interval(1000).pipe(map((_) => ({ data: { hello: 'world' } })));
}
💡 Hint

Decorator @Sse() và interface MessageEvent được import từ @nestjs/common, trong khi Observable, interval, và map được import từ package rxjs.

⚠️ Warning

Các route Server-Sent Events phải trả về một Observable stream.

Trong ví dụ trên, chúng ta đã định nghĩa một route có tên sse cho phép chúng ta truyền các cập nhật real-time. Các event này có thể được lắng nghe bằng EventSource API.

Method sse trả về một Observable phát ra nhiều MessageEvent (trong ví dụ này, nó phát ra một MessageEvent mới mỗi giây). Object MessageEvent nên tuân theo interface sau để khớp với specification:

export interface MessageEvent {
  data: string | object;
  id?: string;
  type?: string;
  retry?: number;
}

Với thiết lập này, giờ đây chúng ta có thể tạo một instance của class EventSource trong ứng dụng phía client, truyền route /sse (khớp với endpoint mà chúng ta đã truyền vào decorator @Sse() ở trên) làm constructor argument.

Instance EventSource mở một kết nối persistent đến HTTP server, server này gửi các event theo định dạng text/event-stream. Kết nối vẫn mở cho đến khi bị đóng bằng cách gọi EventSource.close().

Khi kết nối được mở, các message đến từ server được chuyển đến code của bạn dưới dạng các event. Nếu có trường event trong message đến, event được kích hoạt sẽ giống với giá trị trường event. Nếu không có trường event, thì một event message chung sẽ được phát ra (nguồn).

const eventSource = new EventSource('/sse');
eventSource.onmessage = ({ data }) => {
  console.log('New message', JSON.parse(data));
};
