Compression

Compression có thể giảm đáng kể kích thước của response body, từ đó tăng tốc độ của web app.

💡 Hint

Đối với các website có lưu lượng truy cập cao trong production, việc offload compression ra khỏi application server được khuyến nghị mạnh mẽ — thường là trong một reverse proxy (ví dụ: Nginx). Trong trường hợp đó, bạn không nên sử dụng compression middleware.

Use with Express (default) #

Sử dụng middleware package compression để bật gzip compression.

Trước tiên, cài đặt package cần thiết:

$ npm i --save compression
$ npm i --save-dev @types/compression

Sau khi cài đặt xong, apply compression middleware như global middleware.

import * as compression from 'compression';
// somewhere in your initialization file
app.use(compression());

Use with Fastify #

Nếu sử dụng FastifyAdapter, bạn sẽ muốn sử dụng fastify-compress:

$ npm i --save @fastify/compress

Sau khi cài đặt xong, apply @fastify/compress middleware như global middleware.

⚠️ Warning

Hãy đảm bảo rằng bạn sử dụng type NestFastifyApplication khi tạo application. Nếu không, bạn sẽ không thể sử dụng register để apply compression middleware.

import { FastifyAdapter, NestFastifyApplication } from '@nestjs/platform-fastify';
import compression from '@fastify/compress';

// inside bootstrap()
const app = await NestFactory.create<NestFastifyApplication>(
  AppModule,
  new FastifyAdapter(),
);
await app.register(compression);

Mặc định, @fastify/compress sẽ sử dụng Brotli compression (trên Node >= 11.7.0) khi browser chỉ ra rằng chúng hỗ trợ encoding này. Mặc dù Brotli có thể khá hiệu quả về mặt compression ratio, nhưng nó cũng có thể khá chậm. Mặc định, Brotli đặt maximum compression quality là 11, tuy nhiên nó có thể được điều chỉnh để giảm thời gian compression thay vì chất lượng compression bằng cách điều chỉnh BROTLI_PARAM_QUALITY trong khoảng từ 0 (min) đến 11 (max). Điều này sẽ yêu cầu fine tuning để tối ưu hiệu suất space/time. Ví dụ với quality 4:

import { constants } from 'node:zlib';
// somewhere in your initialization file
await app.register(compression, {
  brotliOptions: {
    params: {
      [constants.BROTLI_PARAM_QUALITY]: 4,
    },
  },
});

Để đơn giản hóa, bạn có thể muốn yêu cầu fastify-compress chỉ sử dụng deflate và gzip để compress response; bạn sẽ nhận được response có thể lớn hơn nhưng chúng sẽ được truyền tải nhanh hơn nhiều.

Để chỉ định encoding, cung cấp argument thứ hai cho app.register:

await app.register(compression, {
  encodings: ['gzip', 'deflate'],
});

Đoạn code trên yêu cầu fastify-compress chỉ sử dụng gzip và deflate encoding, ưu tiên gzip nếu client hỗ trợ cả hai.
