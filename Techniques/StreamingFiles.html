Streaming files

📘 Note

Chương này hướng dẫn cách stream file từ HTTP application của bạn. Các ví dụ được trình bày bên dưới không áp dụng cho ứng dụng GraphQL hoặc Microservice.

Có thể có lúc bạn muốn gửi lại một file từ REST API đến client. Để thực hiện điều này với Nest, thông thường bạn sẽ làm như sau:

@Controller('file')
export class FileController {
  @Get()
  getFile(@Res() res: Response) {
    const file = createReadStream(join(process.cwd(), 'package.json'));
    file.pipe(res);
  }
}

Nhưng khi làm vậy, bạn sẽ mất quyền truy cập vào post-controller interceptor logic. Để xử lý điều này, bạn có thể return một instance StreamableFile và bên dưới, framework sẽ lo việc piping response.

Streamable File class #

StreamableFile là một class giữ tham chiếu đến stream cần được trả về. Để tạo một StreamableFile mới, bạn có thể truyền một Buffer hoặc một Stream vào constructor StreamableFile.

💡 Hint

Class StreamableFile có thể được import từ @nestjs/common.

Cross-platform support #

Fastify, mặc định, có thể hỗ trợ gửi file mà không cần sử dụng StreamableFile. Tuy nhiên, nếu bạn đang sử dụng StreamableFile, Nest sẽ xử lý việc piping stream cho bạn trong cả Express và Fastify.

Example #

Bạn có thể tìm một ví dụ đơn giản về việc trả về package.json dưới dạng file thay vì JSON bên dưới, nhưng ý tưởng này mở rộng một cách tự nhiên cho image, document, và bất kỳ loại file nào khác.

import { Controller, Get, StreamableFile } from '@nestjs/common';
import { createReadStream } from 'node:fs';
import { join } from 'node:path';

@Controller('file')
export class FileController {
  @Get()
  getFile(): StreamableFile {
    const file = createReadStream(join(process.cwd(), 'package.json'));
    return new StreamableFile(file);
  }
}

Content type mặc định (giá trị cho header HTTP response Content-Type) là application/octet-stream. Nếu bạn cần tùy chỉnh giá trị này, bạn có thể sử dụng option type từ StreamableFile, hoặc sử dụng method res.set hoặc decorator @Header(), như sau:

import { Controller, Get, StreamableFile, Res } from '@nestjs/common';
import { createReadStream } from 'node:fs';
import { join } from 'node:path';
import type { Response } from 'express'; // Assuming that we are using the ExpressJS HTTP Adapter

@Controller('file')
export class FileController {
  @Get()
  getFile(): StreamableFile {
    const file = createReadStream(join(process.cwd(), 'package.json'));
    return new StreamableFile(file, {
      type: 'application/json',
      disposition: 'attachment; filename="package.json"',
      // length: 123,
    });
  }

  // Or even:
  @Get()
  getFileChangingResponseObjDirectly(@Res({ passthrough: true }) res: Response): StreamableFile {
    const file = createReadStream(join(process.cwd(), 'package.json'));
    res.set({
      'Content-Type': 'application/json',
      'Content-Disposition': 'attachment; filename="package.json"',
    });
    return new StreamableFile(file);
  }

  // Or even:
  @Get()
  @Header('Content-Type', 'application/json')
  @Header('Content-Disposition', 'attachment; filename="package.json"')
  getFileUsingStaticValues(): StreamableFile {
    const file = createReadStream(join(process.cwd(), 'package.json'));
    return new StreamableFile(file);
  }
}
